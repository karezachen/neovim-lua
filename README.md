## 速查

### 窗口

- 水平分屏 `sh`
- 垂直分屏 `sv`
- 关闭窗口 `sc`
- 关闭其他 `so`
- 窗口跳转 `option` + `h`/`j`/`k`/`l`

### 标签页

- 左右切换标签页 `control` + `h` / `l`
- 关闭标签页 `control` + `w`

### 目录

- 打开 `option` + `m`
- 上下移动 `j`/`k`
- 左右窗口跳转 `option` + `h` / `l`

### 搜索

- 文件搜索 `control` + `p`
- 内容搜索 `control` + `f`
- 搜索结果上下条目跳转 `control` + `j` / `k`
- 搜索结果上下条目跳转 `ESC`, `j` / `k`
